Semana 04

Aquí te voy a contar todo de la semana 04
Proyectos Arduino y programación en CSS.

Introducción

En esta semana hablamos acerca de 3 lenguajes de programación, cómo nos podrá ayudar ChatGpt y algunas páginas web que te ayudan a aprender a programar como jugando.

Clase Virtual

Hay tres lenguajes de desarrollo web que son fundamentales para crear sitios web modernos y funcionales, una de ellas es HTML que define la estructura de tu página web y su contenido, CSS se encarga del diseño y la presentación, y JavaScript se utiliza para añadir interacción y dinamismo. Una forma sencilla de aprenderlo mediante prueba y error es a traves de Liveweave, un editor donde es posible practicar de forma sencilla y en tiempo real.

Ahora si tienes algún problema o se te olvidan los códigos puedes pedir ayuda a ChatGpt, en nuestro caso el profesor nos dió un prompt: "Hola ChatGPT, necesito que seas un desarrollador web y docente para que me ayudes a dar mis primeros pasos en programación web con HTML y CSS. Puedo iniciar a consultarte?" con esto lograrás que Gpt te indique paso a paso qué hacer y te describa el poqué, de esta forma es más sencillo entender cómo funciona todo. Para aprender hay más páginas algunas recomendadas en clase son FLEXBOX FROGGY y FreeCodeCamp

En esta clase tambien ingresamos y aprendimo a usar Netlify, que nos ayudará a alojar y mantener el sitio web que estamos creando. Para ello solo es necesario entrar y crear una cuenta en su página web "https://www.netlify.com/", una vez dentro solo carga los archivos que ya tienes y obtendras la Url para acceder a tu nuevo sitio web.

Para finalizar el profesor nos brindó asesoría por separándonos por grupos e ingrsando a las salas para ver qué problemáticas teníamos.

Clase Presencial

El estilo es algo personal que va cambiando con el tiempo y esto se refleja en todo hasta en la bitácora, por ello es necesario tener todas las herramientas para realizar cualquier cambio. Pero no te compliques, una forma sencilla es copiando alguna otra parte del código que ya contenga el elemento deseado a un nuevo lugar.

En clase practicamos un poco con los cambios y dió tiempo para las preguntas de cualquier detalle a realizar con la bitácora.

Realizamos también un proyecto en clase, que se trataba de reforzar la empatía en los proyectos, para ello tuvimos que hacer un mapa mental a partir de una necesidad, luego ahondamos más y más en las posibles razones y consecuencias que lo puedan haber originado. Era necesario buscar respuestas con Arduino y que sea posible realizarlo en Físico. En nuestro grupo hablamos sobre lo dificil que resulta vivir en soledad y peor aún siendo un adulto mayor. Nuestro compañero tuvo una gran exposición y se explayó de forma genial. Lo mejor de todo es que ¡Ganamos!. Un punto más para el equipo, y vamos por más. c:
