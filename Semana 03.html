Introducción

¿Quieres automatizar todo? Ésta es tu semana, se acabó realizar todos los pasos una y otra vez de forma manual.

Clase Virtual

¿Te imaginas que por cada cambio que realices en la bitácota tengas que cargar los documentos en gihub nuevamente? Imposible, sería muy agotador, por ello lo mejor es vincular visual Studio y Gihub para que cada cambio realizado se refleje en ambos programas automáticamente. Para ello será necesario descargar Git que permite que los repositorios locales sean plenamente funcionales y permite trabajarlos sin conexión o de forma remota con facilidad. Para descargarlo simplemente puedes entrar a: https://git-scm.com/downloads y elegir el sistema que estás usando. Al isntalarlo no te olvides de seleccionar la opción de integrar con con Visual Studio Code.

Acabada la intalación, toca clonar la Url del GiHub previamente copiada desde "Code-HTTPS" y se creará la carpeta donde se va a almacenar todo el proyecto. Aquí usaremos la opción de "Run-Run without Debugging", que ya conocimos previamente. Finalizaremos dirigiendonós a "Terminal" y colocando dos elementos: git config --global user.name "[nombre de usuario]" [ENTER] y git config --global user.mail "[correo integrado]" [ENTER] cambiando los datos solicitados. Siguiendo estos pasos ya tendremos todo sincronizado y listo para editar.

Antes de terminar la clase el profesor brindó asesoría y pidió que compartan su pantalla a aquellos que tenian algun inconveniente con este proceso. Si bien al principio me pasaba el mismo error que a varios, con las correcciones que fue haciendo se solucionó y ya tengo todo listo para trabajar.

Clase Presencial

En esta clase pusimos mayor énfasis en propiedades y códigos en Studio, por ejemplo: ¿Sabías que para poder colocar imágenes es necesario empezar con href="nombre de la carpeta/nombre de la imagen.png"? pero no serviría de nada si no colocas la imagen deseada dentro de la misma carpeta creada para los archivos inicialmente. De igual forma asegúrate de que los nombres esten exactamente igual escritos para que el programa los pueda reconocer.

Si tuvieras problemas al cargar las imágenes puedes subirlas directamente al Gihub, esperas a que aparezca el punto verde y será la señal de que ya ha cargado todo.

Aquí hablamos también de qué son los estilos y usos, que en Studio se refieren a las reglas de estilo CSS aplicadas a un documento HTML y con ellas se puede editar en bloque y administrar dentro del editor. Desde Main Css se puede cambiar el color de estilos h1, h2, etc.

Como todas las clases el profesor nos dejó una actividad, consistía en recolectar la mayor cantidad de soluciones tecnológicas basadas en una o mas Ods. Nosotros como grupo hicimos entre 6 a 7 ideas que incluian desde adopción de animales hasta mejoras en la educación. Posiblemente consideraremos algunas para el proyecto final.
